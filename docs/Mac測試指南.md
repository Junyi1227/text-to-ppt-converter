# 在 Mac 上測試工具（無需 Windows）

## 🎯 情況說明

您在 Mac 上開發，手上沒有 Windows 電腦，但想測試工具功能。

---

## ✅ 最佳方案：在 Mac 上測試 Python 版本

### Python GUI 版本 = Windows .exe 版本

**重要概念**：
- Windows .exe 就是把 Python 程式打包而成
- 功能完全相同
- 介面完全相同
- 只是執行方式不同

所以：**在 Mac 上測試 Python GUI 版本 = 測試 Windows .exe 功能** ✅

---

## 🧪 立即測試（1 分鐘）

### 方法 1：測試 GUI 版本（推薦）

```bash
# 執行 GUI 版本
python3 text_to_ppt_gui.py
```

**您會看到**：
- 視窗開啟
- 文字輸入框（預設有範例文字）
- 「載入文字檔」按鈕
- 「轉換為 PPT」按鈕
- 「清除」按鈕

**測試步驟**：
1. 查看預設範例文字
2. 點「轉換為 PPT」
3. 選擇儲存位置（例如：桌面）
4. 檔名輸入：`測試.pptx`
5. 點儲存
6. 用 PowerPoint/Keynote 開啟檢查結果

### 方法 2：測試命令列版本

```bash
# 快速測試
python3 text_to_ppt.py 範例輸入文字.txt 測試輸出.pptx

# 檢查產生的檔案
open 測試輸出.pptx
```

## 🖥️ 如果真的需要 Windows 環境

### 選項 1：使用虛擬機（最完整）

#### Parallels Desktop（推薦，但付費）
- 官網：https://www.parallels.com/
- 費用：約 $99/年
- 優點：效能最好，整合度高
- 適合：經常需要測試 Windows 程式

#### UTM（免費）
- 官網：https://mac.getutm.app/
- 費用：免費
- 優點：開源、免費
- 缺點：設定較複雜，效能較差

#### VMware Fusion（免費個人版）
- 官網：https://www.vmware.com/products/fusion.html
- 費用：個人使用免費
- 優點：穩定、免費

**安裝步驟**（以 UTM 為例）：
```bash
# 1. 安裝 UTM
brew install --cask utm

# 2. 下載 Windows ISO
# 前往：https://www.microsoft.com/software-download/windows11

# 3. 在 UTM 中建立虛擬機
# 4. 安裝 Windows
# 5. 測試 .exe
```
