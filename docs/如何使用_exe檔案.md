# 如何使用 文字轉PPT工具.exe

## 🎯 這個 .exe 是什麼？

這是一個 **Windows 獨立執行檔**，可以將特定格式的文字自動轉換成 PowerPoint 簡報。

---

## 💻 在 Windows 上使用

### 步驟 1：準備檔案

1. **將以下檔案複製到 Windows 電腦**：
   - `文字轉PPT工具.exe` ⭐ 主程式
   - `Windows用戶使用說明.txt` - 使用手冊
   - `範例輸入文字.txt` - 範例檔案（可選）

2. **建議放在容易找到的位置**：
   - 桌面
   - 或建立專用資料夾（例如：`C:\TextToPPT\`）

---

### 步驟 2：執行程式

1. **雙擊 `文字轉PPT工具.exe`**

2. **第一次執行可能會出現警告**：
   ```
   Windows 已保護您的電腦
   ```
   
   **這是正常的！** 因為執行檔沒有數位簽章。
   
   **解決方法**：
   - 點擊「**更多資訊**」
   - 點擊「**仍要執行**」

3. **程式視窗會開啟** 🎨

---

### 步驟 3：轉換文字為 PPT

#### 方法 A：直接輸入文字

1. 在程式視窗中會看到一個大的文字輸入框
2. 預設已經有範例文字
3. 您可以直接修改，或清空後輸入自己的內容

**文字格式**：
```
##主題標題
這是主題頁面

#內文標題
第一個重點
第二個重點
第三個重點
```

#### 方法 B：載入文字檔

1. 點擊「📂 載入文字檔」按鈕
2. 選擇您的 .txt 檔案
3. 文字會自動載入到輸入框

#### 轉換

1. 點擊「🎨 轉換為 PPT」按鈕
2. 選擇儲存位置和檔名
3. 等待幾秒鐘
4. 完成！ 🎉

---

## 📝 文字格式說明

### 基本規則

- `##` 開頭 → **主題頁面**（藍色背景，大標題，置中）
- `#` 開頭 → **內文頁面**（灰色背景，標題+條列）
- 一般文字 → 加入到前一張投影片

### 完整範例

```
##歡迎使用

#什麼是這個工具？
自動將文字轉換成 PowerPoint
節省製作簡報的時間
確保格式統一

#主要特色
簡單易用
無需安裝 Python
支援自訂格式

##開始使用
立即體驗吧！
```

這會產生：
- 第 1 頁：主題頁（「歡迎使用」）
- 第 2 頁：內文頁（「什麼是這個工具？」+ 3 個重點）
- 第 3 頁：內文頁（「主要特色」+ 3 個重點）
-第 4 頁：主題頁（「開始使用」）

## 🎨 投影片樣式

### 主題頁面（##）
- **背景顏色**：淺藍色
- **字型**：微軟正黑體 44pt 粗體
- **對齊**：置中
- **用途**：章節標題、開場、結尾

### 內文頁面（#）
- **背景顏色**：淺灰色
- **標題**：微軟正黑體 32pt 粗體
- **內文**：微軟正黑體 18pt
- **條列符號**：自動啟用
- **用途**：詳細內容、重點列表

## 🖥️ 在 Mac 上無法使用

**重要**：這個 .exe 檔案**只能在 Windows 上執行**。

如果您需要在 Mac 上使用：
1. 使用 Python 版本：
   ```bash
   python3 text_to_ppt_gui.py
   ```

2. 或使用 VBA 版本（在 PowerPoint 中）

## 💡 使用技巧

### 技巧 1：準備文字模板

建立一個「模板.txt」：
```
##專案報告

#目標
目標一
目標二
目標三

#進度
已完成
進行中
待辦事項

##結論
```

每次使用時複製這個模板並修改內容。

### 技巧 2：快速修改

在程式中直接編輯文字：
- 不需要每次都從檔案載入
- 即時修改，立即轉換

### 技巧 3：批次處理

準備多個 .txt 檔案：
```
報告1.txt
報告2.txt
報告3.txt
```

逐一載入並轉換，快速產生多份簡報。

## ⚠️ 注意事項

### 格式要求
- `##` 和 `#` 必須在行首（前面不能有空格）
- 標記後要有空格或直接換行
- 中英文都支援

### 檔案大小
- .exe 檔案約 20-30 MB（正常）
- 包含了所有必要的程式庫

### Windows Defender
- 第一次執行會出現警告（正常）
- 選擇「仍要執行」即可

## ❓ 常見問題

### Q: 為什麼檔案這麼大？
**A**: 因為包含了 Python 直譯器和所有函式庫，才能獨立執行。

### Q: 需要安裝 PowerPoint 嗎？
**A**: 不需要安裝就能產生 .pptx 檔案，但需要 PowerPoint 或相容軟體才能開啟檢視。

### Q: 可以自訂顏色和字型嗎？
**A**: 目前版本使用固定樣式。如需自訂，請聯絡開發者或使用 Python 版本修改程式碼。

### Q: 支援圖片嗎？
**A**: 目前版本只支援文字。圖片功能可在未來版本加入。

### Q: 在 Mac 上能用嗎？
**A**: 不行，.exe 只能在 Windows 上執行。Mac 請使用 Python 版本。
