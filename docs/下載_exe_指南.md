## ❓ 可能遇到的問題

### 問題：GitHub Actions 失敗（紅色叉叉）

**解決**：
1. 點擊失敗的工作流程
2. 查看紅色錯誤訊息
3. 通常是程式碼錯誤或相依套件問題
4. 修正後重新推送

### 問題：找不到 Artifacts

**原因**：工作流程還在執行中

**解決**：等待出現綠色勾勾

### 問題：.exe 在 Windows 上無法執行

**可能原因**：
- Windows Defender 封鎖
- 缺少運行庫（罕見）

**解決**：
- 點選「更多資訊」→「仍要執行」
- 或右鍵→「以系統管理員身分執行」
