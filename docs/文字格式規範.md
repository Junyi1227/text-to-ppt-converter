# 文字格式規範 v2.0

## 設計理念

針對經文/教學內容，設計簡單直覺的標記語法，支援多種投影片版型。

---

## 基本語法

### 1. 投影片類型標記

使用 `@type:` 來指定投影片類型：

```
@type:title
標題內容

@type:section
章節標題

@type:verse
經文內容

@type:teaching
教導內容

@type:quote
引用段落

@type:list
條列重點

@type:twoColumn
左右兩欄內容
```

### 2. 分隔符號

- `---` (三個破折號) = 分隔不同投影片
- `===` (三個等號) = 分隔同一投影片的不同區塊

---

## 投影片類型詳解

### 📘 @type:title - 封面標題頁
```
@type:title
主標題
===
副標題
===
日期或作者
```

**效果：**
- 居中大字標題
- 藍色漸層背景
- 三層資訊（主標題/副標題/日期）

---

### 📖 @type:section - 章節分隔頁
```
@type:section
第一部分：避難所
```

**效果：**
- 深色背景
- 大號白字
- 用於章節切換

---

### 📜 @type:verse - 經文引用頁
```
@type:verse
〈創 19:17〉
===
領他們出來以後，就說：「逃命吧！
不可回頭看，也不可在平原站住。
要往山上逃跑，免得你被剿滅。」
```

**效果：**
- 經文出處（小字、上方）
- 經文內容（大字、居中）
- 特殊經文字型樣式

---

### 💡 @type:teaching - 教導內容頁
```
@type:teaching
主要教導
===
• 第一個重點
• 第二個重點
• 第三個重點
```

**效果：**
- 標題（粗體、左上）
- 內容條列（項目符號）
- 淺色背景

---

### 💬 @type:quote - 引用段落頁
```
@type:quote
人只要做足以承擔的事就好。
大患難來臨時，
應該要按照我耶和華的引導前往避難所。
```

**效果：**
- 引用框樣式
- 斜體字
- 特殊背景色

---

### 📝 @type:list - 條列重點頁
```
@type:list
避開的事物
===
• 因宇宙運行法則而發生的痛苦
• 大患難和戰爭
• 超出承擔能力的事情
```

**效果：**
- 清晰的標題
- 大號項目符號
- 層次分明

---

### ⚖️ @type:twoColumn - 兩欄對照頁
```
@type:twoColumn
若學習
當天就能瞭解
===
若不學習
一輩子都不會瞭解
```

**效果：**
- 左右對照
- 適合對比教學
- 兩欄獨立框線

---

## 完整範例

```
@type:title
2025年12月31日話語
===
神的避難所
===
2025.12.31

---

@type:section
第一部分：學習與領悟

---

@type:twoColumn
若學習
當天就能瞭解
===
若不學習
一輩子都不會瞭解

---

@type:teaching
宇宙法則的重要性
===
• 廣闊宇宙或地球內萬有法則的運行，是不能中斷的
• 人必須避開因此發生的極端狀況才能存活
• 人只要做足以承擔的事就好

---

@type:verse
〈創 19:17〉
===
領他們出來以後，就說：「逃命吧！
不可回頭看，也不可在平原站住。
要往山上逃跑，免得你被剿滅。」

---

@type:verse
〈太 2:13-14〉
===
他們去後，有主的使者向約瑟夢中顯現，說：
「起來！帶着小孩子同他母親逃往埃及，
住在那裏，等我吩咐你；
因為希律必尋找小孩子，要除滅他。」

---

@type:quote
在極端危急的時候，要倚靠神，
躲到神引導的地方，
而且要在那裡實踐相關的旨意。

---

@type:list
應該要避開的事物
===
• 因我耶和華創造的宇宙運行法則而發生的痛苦
• 大患難和戰爭
• 超出承擔能力的爭戰

---

@type:section
第二部分：義的腦內啡

---

@type:teaching
如何啟動義的腦內啡
===
• 要用希望來啟動義的腦內啡
• 要讓義的腦內啡在內心、想法、精神中都滿溢出來
• 要喜悅並興奮地展開神的旨意

---

@type:verse
〈該 1:14〉
===
耶和華激動猶大省長撒拉鐵的兒子所羅巴伯
和約撒答的兒子大祭司約書亞，
並剩下之百姓的心，
他們就來為萬軍之耶和華－他們神的殿做工。

---

@type:quote
應該要興奮起來做神的事工。
在展開神的旨意時要喜悅並興奮，
腦內啡才會如同火山那般爆發，
不會中斷，所以每天都會喜悅，
一輩子都會如此。阿們！

---

@type:list
營造興奮的生活
===
• 不要為了尋求刺激而喝酒、抽菸、吸毒、性交
• 不要動武，而是要依循理致自然地做
• 要如季節更迭般順理地做才會萬事亨通
• 要讓義的腦內啡發揮作用
```

## 向後相容

為了相容舊版本，仍然支援簡易標記：

```
## 主標題    → 自動轉換為 @type:title
# 小標題     → 自動轉換為 @type:teaching
普通文字     → 作為內容
```

---

## 進階選項（可選）

### 自訂樣式標記

```
@type:verse
@style:large
〈詩篇 23〉
===
耶和華是我的牧者...
```

可用選項：
- `@style:large` - 大字體
- `@style:center` - 居中
- `@style:bold` - 粗體
- `@style:color:blue` - 藍色文字

---

## 建議的 Word 標記方式

在 Word 中準備內容時，建議使用顏色標記：

| 顏色 | 用途 | 自動轉換 |
|------|------|---------|
| **藍色** | 經文出處 | `@type:verse` |
| **綠色** | 教導重點 | `@type:teaching` |
| **紅色** | 章節標題 | `@type:section` |
| **紫色** | 引用段落 | `@type:quote` |

這樣提取時可以自動判斷類型！

## 總結

### ✅ 優點
1. **簡單直覺** - `@type:` 清楚標示類型
2. **易於編輯** - 純文字格式，任何編輯器都可用
3. **結構清晰** - `---` 分隔投影片，`===` 分隔區塊
4. **專為經文設計** - 有專門的 verse、teaching、quote 類型
5. **可擴展** - 未來可以輕鬆新增更多類型

### 📝 使用流程
1. Word 中用不同顏色標記內容
2. 提取工具自動識別顏色 → 轉換成 @type
3. 手動調整格式（如果需要）
4. 轉換成 PPT
