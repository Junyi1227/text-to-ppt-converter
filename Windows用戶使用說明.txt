======================================================
文字轉 PowerPoint 工具 - Windows 使用說明
======================================================

🎉 歡迎使用！這個工具可以快速將文字轉換成 PowerPoint 簡報。

⭐ 特點：無需安裝 Python 或其他軟體，雙擊即可使用！

======================================================

📋 快速開始（三步驟）
--------------------

步驟 1：開啟程式
  • 雙擊「文字轉PPT工具.exe」
  • （第一次執行可能需要幾秒鐘）

步驟 2：輸入或載入文字
  • 方法 A：直接在視窗中輸入或貼上文字
  • 方法 B：點擊「📂 載入文字檔」選擇 .txt 檔案

步驟 3：轉換
  • 點擊「🎨 轉換為 PPT」
  • 選擇儲存位置
  • 完成！


📝 文字格式說明
---------------

基本規則：
  ##主題標題        ← 建立主題頁面（藍色背景，大標題）
  主題內容說明      ← 可選的內容
  
  #內文標題         ← 建立內文頁面（灰色背景）
  第一個重點        ← 自動變成條列項目
  第二個重點        ← 自動變成條列項目
  第三個重點        ← 自動變成條列項目

完整範例：

##專案簡介
歡迎參加本次專案說明會

#專案目標
提升使用者體驗
增加系統效能
降低維護成本

#預期成果
使用者滿意度提升 30%
系統回應時間減少 50%
維護成本降低 40%

##總結
感謝您的參與！

🎨 投影片樣式
-------------

主題頁面（## 標記）：
  • 背景：淺藍色
  • 字型：微軟正黑體 44pt 粗體
  • 對齊：置中
  • 用途：章節標題、開場、結尾

內文頁面（# 標記）：
  • 背景：淺灰色
  • 標題：微軟正黑體 32pt 粗體
  • 內文：微軟正黑體 18pt
  • 條列：自動顯示項目符號
  • 用途：詳細內容、重點列表


⚠️ 注意事項
-----------

1. 格式要求：
   • ## 和 # 必須在行首（前面不能有空格）
   • 每個標記後面要有一個空格
   • 例如：「##標題」或「## 標題」都可以

2. Windows Defender 可能會警告：
   • 第一次執行時，可能顯示「Windows 已保護您的電腦」
   • 點選「更多資訊」→「仍要執行」
   • 這是正常的，因為執行檔沒有數位簽章

3. 檔案大小：
   • 執行檔約 20-30 MB
   • 這是正常的，因為包含了所有必要的程式庫

4. 字型問題：
   • 如果產生的 PPT 字型不正確
   • 確認您的電腦已安裝「微軟正黑體」
   • 或修改程式（聯絡開發者）


❗ 疑難排解
----------

問題 1：程式無法開啟
解決方法：
  • 確認是否已解壓縮（不能直接在壓縮檔中執行）
  • 嘗試「以系統管理員身分執行」
  • 檢查防毒軟體是否封鎖

問題 2：轉換後的 PPT 無法開啟
解決方法：
  • 確認已安裝 Microsoft PowerPoint
  • 或使用 LibreOffice Impress（免費）
  • 檢查檔案是否完整儲存（不要在儲存過程中關閉程式）

問題 3：投影片格式不正確
解決方法：
  • 檢查文字格式（## 和 # 的位置）
  • 確認沒有多餘的空格或特殊字元
  • 參考範例檔案的格式

問題 4：程式執行很慢
解決方法：
  • 第一次執行會比較慢（正在解壓縮）
  • 後續執行會快很多
  • 如果持續很慢，可能是電腦效能問題

問題 5：中文字顯示不正確
解決方法：
  • 確認文字檔使用 UTF-8 編碼
  • 用記事本開啟 → 另存新檔 → 編碼選擇「UTF-8」


🔐 安全性說明
