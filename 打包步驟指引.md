## 📋 接下來的步驟

### 步驟 1：在 GitHub 建立 Repository（2 分鐘）

1. **開啟瀏覽器，前往 GitHub**
   ```
   https://github.com/new
   ```

2. **填寫 Repository 資訊**
   - **Repository name**: `text-to-ppt-converter`（或您喜歡的名稱）
   - **Description**: `文字轉 PowerPoint 工具`
   - **選擇 Public** ✅ （才能免費使用 GitHub Actions）
   - **不要勾選** "Add a README file" ❌
   - **不要勾選** "Add .gitignore" ❌
   - **不要勾選** "Choose a license" ❌

3. **點擊綠色按鈕 "Create repository"**

4. **記下您的 GitHub 帳號和 Repository 名稱**
   - GitHub 帳號：`_______________`
   - Repository 名稱：`text-to-ppt-converter`

---

### 步驟 2：執行自動設定腳本（3 分鐘）

回到終端機（Terminal），執行：

```bash
bash 快速開始.sh
```

**腳本會詢問您**：
1. GitHub 帳號名稱
2. Repository 名稱（直接按 Enter 使用預設值）
3. 您的名字（用於 Git commit）
4. 您的 Email

**然後腳本會自動**：
- 初始化 Git
- 加入所有檔案
- 建立第一個 commit
- 推送到 GitHub

---

### 步驟 3：處理推送認證（如果需要）

**如果推送時要求輸入帳號密碼**：

#### 選項 A：使用 Personal Access Token（推薦）

1. **產生 Token**
   - 前往：https://github.com/settings/tokens
   - 點擊 `Generate new token` → `Generate new token (classic)`
   - Note: 輸入 `text-to-ppt-converter`
   - Expiration: 選擇 `90 days` 或 `No expiration`
   - 勾選權限：✅ `repo` （勾選整個 repo 區塊）
   - 往下捲動，點擊 `Generate token`
   - **立即複製 token**（只會顯示一次！）

2. **在終端機輸入**
   - Username: 您的 GitHub 帳號
   - Password: 貼上剛才複製的 token

#### 選項 B：使用 SSH（一次設定，永久使用）

```bash
# 1. 產生 SSH 金鑰
ssh-keygen -t ed25519 -C "您的email@example.com"
# 按 Enter 使用預設路徑
# 按 Enter 跳過密碼（或設定密碼）

# 2. 複製公鑰
cat ~/.ssh/id_ed25519.pub

# 3. 前往 GitHub 加入 SSH 金鑰
# https://github.com/settings/ssh/new
# Title: Mac
# Key: 貼上剛才複製的內容
# 點擊 Add SSH key

# 4. 修改 remote URL
git remote set-url origin git@github.com:您的帳號/text-to-ppt-converter.git

# 5. 重新推送
git push -u origin main
```

---

### 步驟 4：等待打包完成（3-5 分鐘）

1. **前往 GitHub Actions**
   ```
   https://github.com/您的帳號/text-to-ppt-converter/actions
   ```

2. **查看工作流程狀態**
   - 🟡 黃色圓圈：正在執行中
   - ✅ 綠色勾勾：成功完成
   - ❌ 紅色叉叉：失敗（查看錯誤訊息）

3. **即時查看進度**
   - 點擊工作流程名稱
   - 可以看到即時日誌

---

### 步驟 5：下載 .exe 檔案

**當工作流程完成後（綠色勾勾）**：

1. 點擊完成的工作流程
2. 往下捲動到 "Artifacts" 區域
3. 會看到：
   - `Windows-GUI-Executable` - GUI 版本 🎨
   - `Windows-CLI-Executable` - 命令列版本 💻
4. 點擊 `Windows-GUI-Executable` 下載
5. 解壓縮 .zip 檔案
6. 就會看到 `文字轉PPT工具.exe` 🎉

## ❗ 可能遇到的問題

### 問題 1：推送時出現 "remote: Repository not found"

**原因**：Repository 尚未建立或 URL 錯誤

**解決**：
1. 確認已在 GitHub 建立 repository
2. 檢查 repository URL 是否正確

### 問題 2：GitHub Actions 失敗

**原因**：檔案缺失或程式碼錯誤

**解決**：
1. 點擊失敗的工作流程
2. 查看紅色錯誤訊息
3. 修正問題後重新推送

### 問題 3：找不到 Artifacts

**原因**：工作流程還在執行或已失敗

**解決**：
- 等待工作流程完成（綠色勾勾）
- 如果失敗，查看錯誤並修正
